# How to commit and push changes:

# Step 1: Navigate to your local repository
cd path/to/your/jekyll/project

# Step 2: Pull the latest changes from GitHub
git pull origin main

# Step 3: Check the status of your repository
git status

# Step 4: Add all modified files to the staging area
git add .

# Step 5: Commit your changes with a descriptive message
git commit -m "Your commit message describing the changes"

# Step 6: Push your changes to the GitHub repository
git push origin main
