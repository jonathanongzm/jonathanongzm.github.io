Jonathan Z. Ong - Developmental Cognitive Neuroscience ======================================================

Welcome to the personal website of Jonathan Z. Ong, a master's student specializing in developmental cognitive neuroscience. This website showcases my research, including studies on infant neurodevelopment, parent-infant interactions, and optogenetic models, as well as my academic contributions and publications.

Installation and Setup
----------------------
To run this website locally, you will need to have Jekyll installed. Follow these steps:

1. Clone the repository:
   - `git clone https://github.com/jonathanongzm/your-repository-name.git`
   - `cd your-repository-name`

2. Install dependencies:
   - `bundle install`

3. Run the website locally:
   - `bundle exec jekyll serve`

4. Open your browser and go to `http://localhost:4000` to see the website.
